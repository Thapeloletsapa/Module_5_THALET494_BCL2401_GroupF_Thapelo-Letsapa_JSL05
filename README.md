# [JSL05] Submission: Galaxy Playlist Generator


## Playlists App
### Description
  This is a simple application that allows users to generate playlists based
  on their preferred music genre. The user can select their preferred genre,
  and the application will generate a playlist of songs that match the genre.
  The playlist can then be displayed to the user.

### Technologies Used
- JavaScript
- HTML
- CSS
- Node.js (optional)


This challenge will test your ability to manipulate arrays and objects, use the `map()` function for data transformation, and dynamically update the DOM based on data. Good luck, and may your mix be truly awesome!
